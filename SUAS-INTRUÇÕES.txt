Segue os passos importantes para entender a linha de raciocínio na solução.

1 - Cria uma tabela para obter os dados com o número de campos que o arquivo de log oferece. 
2 - Os dados do arquivo foram formatados manualmente para realizar o UPLOAD na tabela (INSERT). 
3 - Obs .: Como o arquivo de log não possui uma separação de campos lógica (campos separados por ";", "TAB", ",", ETC.), não foi feita uma pesquisa automática de um arquivo "TXT". O que está interessado em automatizar a leitura dos resultados dos novos arquivos de LOG (O ideal, seria o conjunto de dados do arquivo de registro para uma formatação padrão). 
4 - Criamos uma consulta Recursiva para gerar os pilotos e uma consulta para gerar os resultados dos resultados solicitados, evitando criação de novos campos em tabela ou de procedimento apenas para exibição de dados solicitados. 
5 - Com apenas uma consulta disponível, basta utilizar em qualquer um dos sistemas diretos ou de outra forma dentro de uma visão para os dados solicitados.
